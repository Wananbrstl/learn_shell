# default variable

| 变量   | 含义    |
|--------------- | --------------- |
| $#   | 传入shell文件参数的个数   |
| $*   | 传入shell文件的参数，以`"a b c d"`的形式输出   |
| $@   | 传入shell文件的参数，以`"a" "b" "c" "d"`的形式输出   |
| $$   | 当前可执行文件的运行进程号   |
| $?   | 上一条命令的退出状态（注意不是stderr,而是exit code！）0表示正常，其他表示错误   |
| $(command)   | 返回这条指令的stdout（可嵌套）   |
| $`command`   | 返回这条指令的stdout（不可嵌套）   |
